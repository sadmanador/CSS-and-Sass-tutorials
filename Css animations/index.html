2D Transformation

Transform, below properties are necessary to learn

  1. Translate: moves the element from it's origin
  2. rotate: rotates elements from its origin
  3. scale: either increase or decrease the size of the element
  4. scaleX: increase or decrease the width
  5. scaleY: increase or decrease the length
  6. skewX: shew things in Y-xis
  7. skewY: shew things in y-axis
  8. skew: in both axis. skew receives a degree value
  9. matrix: a combine technique of transformation. it take 6 values. scaleX, skewX, scaleY, translateX,translateY
