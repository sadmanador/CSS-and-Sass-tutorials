Gradient: linear-gradient, radial-gradient, and conic- gradient.

Default gradient

  • backgroung-image: linear-gradinet(red, Yellow, green)

Grident flow side to side

  • background-image: linear-gradient(to right, red, Yellow, green)

Grident angel

  • background-image: linear-gradient(to bottom right, red, Yellow, green)

Gradient angel with degree

  • background-image: linear-gradient(180deg, red, Yellow, green)

Radial gradient

  • radial gradient: is king of gradient which start from the center
  • background-image: radial-gradient(red, Yellow);

Conic-gradient

  • conic gradient surrounds the center and move along the perifery
  • background-image: conic-gradient(red, Yellow);
