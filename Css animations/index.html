2D animations

Transform, below properties are necessery to learn

  • Translate: moves the element from it's origin
  • rotate: rotates elements from its origin
  • scale: either increase or decrease the size of the element
  • scaleX: increase or decrease the width
  • scaleY: increase or decrease the lenght
  • skewX: shew things in Y-xis
  • skewY: shew things in y-axis
  • skew: in both axis. skew recives a degree value
  • matrix: a combine technique of transformation. it take 6 values. scaleX, skewX, scaleY, translateX,translateY
